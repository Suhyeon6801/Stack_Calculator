# Stack_Calculator


### 구현 내용
- 실수 계산식을 입력받아서 결과값을 출력(소수 셋째자리에서 반올림)하는 계산기를 구현한다.
- “EOI”라는 입력을 받으면 계산을 끝낸다.
- 괄호의 균형이 맞지 않거나 0.0으로 나누는 경우 에러와 그에 상응하는 문구를 출력한다.
